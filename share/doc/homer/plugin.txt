homer plugin NAME [options]

Installs (or uninstalls) plugins for ZSH. Essentially a
lightweight command-line interface for Antigen, this command manages its
dependencies in your ~/etc/plugins.zsh file and can remove dependencies
as well.

OPTIONS:

  -r    Remove this plugin from ~/etc/aliases.zsh instead of adding it.

  -m MESSAGE    Customize the Git message for this action.
