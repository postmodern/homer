Usage: homer COMMAND [ARGUMENTS]

Homer is your home directory manager. It helps manage things like
plugins, aliases, and dotfiles. For more information on this software,
type `man homer`.

Commands:

- alias     Save an alias in ~/etc/aliases.zsh.
- grep      Search Homer shared source and your own configs simultaneously
- help      View this and other command help.
- init      Initialize a home directory. Use -c to clone an existing
            home directory repo.
- plugin    Install a plugin with Antigen to ~/etc/plugins.zsh.
- save      Commit changes to the repo and push to origin.
- script    Save a bin script to the repo.
- update    Update the repo and stash current changes.
- version   Print the current version of Homer

Type `homer help COMMAND` to get help on a particular command.
