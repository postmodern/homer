homer upgrade [options]

Upgrade to the latest version of Homer. Installs templates to the home
directory if they are newer than the ones already there, and can
optionally use Homebrew to update the codebase.

OPTIONS:

  -p    Use a package manager (Homebrew) to upgrade.
  -c    Commit changes to the repo immediately.
  -m    Optionally supply a message when committing.
